# UNIKER GitHub Practice

This is a safe practice repository for learning the GitHub workflow with Codex.

## Goal

- Make a small local change.
- Commit the change.
- Push it to GitHub.
- Pull a remote change.
- Merge a feature branch.

## Codex prompt to try

Read this repository and explain the current Git status.
Do not edit files yet.
Return the result as:

- current branch
- changed files
- latest commit
- next safe Git action
